# Files #

To setup:
> npm install

To run:
> node app.js

You can now access the URL `http://localhost:3000/api/ls?path=.` and you should see stuff.

Also try:
`http://localhost:3000/api/cat?path=app.js`

### The GUI
..is taking shape at http://localhost:3000/
